Syntax Highlighting for C0 in Pygments

Enable Syntax Highlighting for Frank Pfenning's imperative teaching language C0 in Pygments.

The package provides

  • style unitue
  • a lexer for the C0-language

Installation

Installation via 

sudo python setup.py install

Don't forget install Pygments first. A simple

sudo easy_install Pygments

should do it.
